# Dictionary App - React

A modern, responsive Dictionary application built with React. This app allows users to search for word definitions, listen to pronunciations, and explore synonyms with a clean and animated user interface.

## üåü Features

-   **Word Definitions**: Detailed meanings and examples.
-   **Audio Pronunciation**: Listen to how words are pronounced.
-   **Synonyms**: Explore related words.
-   **Typewriter Animation**: Dynamic homepage heading.
-   **Modern UI**: 
    -   Centered search experience.
    -   Smooth fade-in and slide-up animations for results.
    -   Dark/Light mode support.
    -   Clean result page with auto-hiding search bar.
-   **Responsive Design**: Optimized for desktop and mobile.

## üõ† Tech Stack

-   **Frontend**: React, Material-UI, Semantic UI React
-   **Animation**: Typewriter Effect, CSS Keyframes (`slideUp`, `pulse`)
-   **API**: [Free Dictionary API](https://dictionaryapi.dev/)
-   **Icons**: FontAwesome, Material Icons, Semantic UI Icons

## üöÄ Installation & Run

1.  **Clone the repository**
    ```sh
    git clone https://github.com/ajf013/dictionary-app-react.git
    cd dictionary-app-react
    ```

2.  **Install dependencies**
    ```sh
    npm install --legacy-peer-deps
    ```

3.  **Start the application**
    ```sh
    npm start
    ```
    *Note: If you encounter OpenSSL errors (Node 17+), use:*
    ```sh
    NODE_OPTIONS=--openssl-legacy-provider npm start
    ```

4.  **Open in Browser**
    Visit `http://localhost:3000` (or the port shown in your terminal).
